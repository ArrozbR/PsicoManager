# PsicoManager 🧠💻

Bem-vindo ao repositório do **PsicoManager**, um sistema de gestão voltado para clínicas de psicologia. 

O objetivo principal deste projeto é desenvolver uma plataforma Web que otimize a rotina administrativa e clínica, garantindo a segurança das informações dos pacientes em total conformidade com a **Lei Geral de Proteção de Dados** (LGPD).

**Link do Drive:** [Clique Aqui](https://drive.google.com/drive/folders/1j6rBFBWW8qcbF5BU6HAJt7rIDnYt5lyo?usp=sharing)

---

## 🎯 Escopo do Sistema

O sistema cobrirá as seguintes atividades essenciais da clínica:

* **Gestão de Consultas:** Agendamento, cancelamento e confirmação de consultas.
* **Comunicação Automatizada:** Envio de lembretes e avisos aos pacientes referentes a sessões e pendências financeiras.
* **Prontuário Eletrônico:** Registro e armazenamento seguro da evolução clínica dos pacientes.
* **Controle Financeiro:** Gerenciamento de pagamentos, faturamento e controle de inadimplência.
